创建群组——pengwt

url链接：http://mt.wujie.com.cn/api/groupchat/create

post

参数 值
owner 群主uid(必)
groupname 群组名称(必)
desc 群组描述(必)
activity_id 活动id
avatar 头像url
members 群组成员uid(必) 增加或删除一个成员: ['11242'] 增加或删除多个成员: ['11242','12536']
public 是否是公开群, 此属性为必须的,为false时为私有群
maxusers 群组成员最大数(包括群主), 值为数值类型,默认值200,此属性为可选的
approval 加入公开群是否需要批准, 默认值是true

参数说明（返回参数可补充）

参数 说明
groupid 群组id
groupname 群组名称
avatar 群组头像
description 群组描述
owner 管理员uid
type 类型 0:私有群 1:活动群 2:商机群 3:网点群
activity_id 活动id
opportunity_id 商机id
memberinfo.uid 群组成员uid
memberinfo.username 群组成员用户名
memberinfo.avatar 群组成员头像
memberinfo.nickname 群组成员昵称
number 群人数
