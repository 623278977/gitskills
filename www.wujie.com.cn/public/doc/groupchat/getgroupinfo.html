参数说明（返回参数可补充）

参数 说明
groupid 群组id
sendtime 发消息时间
owner 管理员uid
type 类型 0:私有群 1:活动群 2:商机群 3:网点群
activity_id 活动id
opportunity_id 商机id
number 群组成员人数
avatar 群组头像
groupname 群组名称
created_at 创建时间
updated_at 更新时间
description 群组描述
members.uid 群组成员uid
members.username 群组成员用户名
members.uid 群组成员uid
members.avatar 群组成员头像
members.nickname 群组成员昵称
