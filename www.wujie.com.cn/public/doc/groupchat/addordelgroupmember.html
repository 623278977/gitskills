群组加人/减人 从群组退出——pengwt

url链接：http://mt.wujie.com.cn/api/groupchat/addordelmember

post

参数 值
groupid 群组id(必)
act 动作(必) add:加人;delete:减人
member_uid 群组成员uid(必) 增加或删除一个成员: ['11242'] 增加或删除多个成员: ['11242','12536']

参数说明（返回参数可补充）

参数 说明
groupid 群组id
groupname 群组名称
avatar 群组头像
description 群组描述
owner 管理员uid
type 类型 0:私有群 1:活动群 2:商机群 3:网点群
activity_id 活动id
opportunity_id 商机id
memberinfo.uid 群组成员uid
memberinfo.username 群组成员用户名
memberinfo.avatar 群组成员头像
memberinfo.nickname 群组成员昵称
number 群人数
