参数说明（结构改了）

参数 值
thisMonth(本月订单)、lastMonth(上月订单)、
beforeOrder(之前订单、fail_order(失败订单)
id 订单id
pay_way 支付方式
deadline 订单过期时间
score_price 积分形式的价格
created_at 创建时间
type 类型:活动(activity)，直播(live)，录播(video)，品牌加盟定金(brand)， 品牌商品加盟(brand_goods)，资讯(news)，积分充值(score)
order_type 订单类型
name 订单相应标题、名称
price 订单非积分价格
pay_at 支付时间
count
thisMonth 本月订单数量
lastMonth 上月订单数量
beforeOrder 之前订单数量
fail 失败订单数量
