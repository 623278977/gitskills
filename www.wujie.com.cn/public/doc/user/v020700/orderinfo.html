订单详情——yaokai

url链接：http://mt.wujie.com.cn/api/user/myorderinfo/_v020700

post

参数 值
uid（必） 用户uid
type 类型 (type和refer_id都有的情况下,获取订单数据)
refer_id 关联id (type和refer_id都有的情况下,获取订单数据)
order_no 订单号 (只有order_no,获取全部的订单详情)

参数说明（返回参数可补充）

参数 值
(新)type_name 商品类型
(新)order_type 订单类型
(新)资讯相关orderInfo.type 为news时
id 资讯id
name 资讯标题
summary 资讯描述
logo 资讯logo
author 资讯作者
view 浏览量
count_zan 点赞数
count_comment 评论数
price 订单非积分价格
pay_at 支付时间
活动相关orderInfo.type 为activity时
id 活动id
name 活动标题
list_img 活动图片
keywords 关键字
begin_time_format/begin_time 活动开始时间
activity_status 活动状态：0未开始,1进行中,2已结束
cities 城市
直播相关orderInfo.type 为live时
id 活动id
name 直播标题
list_img 直播图片
summary 直播描述
keywords 关键字
begin_time_format/begin_time 活动开始时间
status 直播状态：0未开始,1进行中,2已结束
score_price 最小积分形式价格
品牌相关orderInfo.type 为 brand 时
id 品牌id
name 品牌名称
goods_name 品牌商品(套餐)名称
goods_league 品牌商品(套餐)描述
logo 品牌logo
keywords 关键字
category_name 分类
treaty 合约
investment_arrange 投资额度
id 关联项目的id

begin_time 开始时间 (活动,直播,点播特有字段)
begin_time_format 开始时间 (活动,直播,点播特有字段)
cities 城市 (活动特有字段)
detail 图文详细 (品牌特有字段)
treaty 合约 (品牌特有字段)
investment_arrange 投资额度 (品牌特有字段)
category_name 分类 (品牌特有字段)
length 播放时长 (点播特有字段)
image 列表图片 (点播特有字段)
