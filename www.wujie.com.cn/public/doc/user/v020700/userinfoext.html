用户附加信息——tangjb

url链接：http://mt.wujie.com.cn/api/user/userinfoext/_v020700

post

参数 值
uid（必） 用户uid

返回数据(例):

{
    "message": {
        "currency": 5778,
        "score": 101730,
        "share_count": 9,
        "read_count": 7,
        "intend_count": 0,
        "currency_total": "10",
        "is_done_invitecode": 0,
        "unread_messages": 0,
        "invite_count": 0,
        "is_sign": 0,
        "serial_sign": 0,
        "sign_score": 10,
        "extracted": 0,
        "invite_sign_count": 0
    },
    "status": true,
    "forwardUrl": ""
}

参数说明（返回参数可补充）

参数 值
currency 佣金
score 当前积分
share_count 分享次数
read_count 阅读次数
intend_count 意向客户
currency_total 累积佣金
is_done_invitecode 是否已填写邀请码 1 :是 0 :否
invite_count 填写我的邀请码的人数
is_sign 今天是否签到，0或1
serial_sign 已经连续签到几次
sign_score 本次可赠送
extracted 已经提取佣金
unread_messages 未读消息
invite_sign_count 累计邀请好友报名
