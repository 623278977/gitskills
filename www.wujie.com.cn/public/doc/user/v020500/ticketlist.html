我的票券，未完成的票券—pengwt

url链接：http://mt.wujie.com.cn/api/user/userticketlist

post

参数 值
uid(必) 用户uid
type(必) 我的票券type 传my 未完成的type传notover
page 分页从1开始的整数
pageSize 每页几条数据默认15

返回数据(例):

{ "message": [ { "group": "expire", "subject": "温州楼塌悲剧，只怪房主自私自利吗？-杭州站", "begin_time": "2016年10月12日 09:00", "ticket_type": "现场票", "price": "888", "online_money": "888.00", "ticket_name": "", "ticket_status": "超出支付时间", "activity_id": 252, "is_sign": 0, "ticket_url": "http://dev.wjsq3.loc/webapp/ticket/not-ticket?id=1512&pagetag=02-2", "is_over": 1, "order_lefttime": -7950325, "maker_id": 98, "live_id": 173, "video_id": 0, "ticket_id": 1512, "is_check": 0, "pay_way":"微信支付", "order_id":1235 } ], "status": true, "forwardUrl": "" }

参数 值
group 分组名称 starting :正在进行 no_start:还未开启 end:已过期 need_pay:请尽快支付 expire:已失效
order_no 订单号
subject 活动标题
ticket_id 门票id
ticket_type 门票类型
ticket_name 门票名称
ticket_status 门票状态
price 门票价格(总)
online_money 需要支付的门票价格
begin_time 活动开始时间（字符串：2016年10月12日 09:00）
surplus_time 还差多少秒开始直播，为-1表示已经开始或结束
maker_id 会场id
live_id 直播id
activity_id 活动id
video_id 点播id
is_over 活动是否结束
is_check 门票是否验证 1 已验证 0未验证
order_status 订单状态-2：删除，-1：超时，0：待支付，1：已支付
order_lefttime 订单状态是未支付情况下:还剩多少秒过期 返回的秒数
ticket_url 我的门票详情url
pay_way 支付方式
order_id 订单id
activity_ticket_id 活动门票id
