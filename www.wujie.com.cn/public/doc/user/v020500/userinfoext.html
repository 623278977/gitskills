用户附加信息——pengwt

url链接：http://mt.wujie.com.cn/api/user/userinfoext/_v020500

post

参数 值
uid（必） 用户uid

返回数据(例):

{ "message": { "currency": 0, "share_count": 0, "intend_count": 0, "currency_total": 0 }, "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 值
currency 佣金
share_count 分享次数
read_count 阅读次数
intend_count 意向客户
currency_total 累积佣金
is_done_invitecode 是否已填写邀请码 1 :是 0 :否
invite_count 填写我的邀请码的人数
is_sign 今天是否签到，0或1
serial_sign 已经连续签到几次
sign_score 本次可赠送
extracted 已经提取佣金
unread_messages 未读消息
