我的订单——pengwt

url链接：http://mt.wujie.com.cn/api/user/myorders/_v020500

post

参数 值
uid（必） 用户uid
page 分页(默认1)
page_size 分页长度(默认10)
is_complete 1:已完成订单(默认值) 0:未完成订单

返回数据(例):

参数说明（返回参数可补充）

参数 值
status 订单状态 已支付,未支付,已过期,删除
amount 支付金额
type 订单类型 brand:品牌 ,activity:活动 ,live:直播 ,video:点播
type_name 订单类型,用于页面展示
oi_id 订单认购项id (直播,活动,点播为ticket_id)
pay_way 支付方式
pay_at 支付时间
goodsName 商品名称
refer_id 关联的id
treaty 线上合约 (类型为brand时返回)
deadline (未完成订单的特有字段) 过期时间(单位秒) -1:已过期 非-1的为距离过期的秒数
created_at (未完成订单的特有字段) 订单生成时间
id 订单id
order_no 订单号（未完成订单的特有字段）
price 门票单价（未完成订单的特有字段，限门票类数据）
ticket_id 门票ID（未完成订单的特有字段，限门票类数据）
ticket_type 门票类型（未完成订单的特有字段，限门票类数据，1现场票，2直播票，-1点播票）
oi_type 品牌加盟类型（当type为brand或者brand_goods时的特有字段 brand品牌加盟 brand_goods品牌详情页购买）
user_ticket_id 用户购买票券id（已经完成订单特有字段）
brand_goods_id 品牌下的商品id（品牌特有字段）
price 原价
