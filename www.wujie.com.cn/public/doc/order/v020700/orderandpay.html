使用积分下单并支付——tangjb

url链接：http://mt.wujie.com.cn/api/order/order-and-pay/_v020700

post

参数 值
uid(必) 当前登录用户uid
items(必) 
            购买项，一个二维数组
            [
                ['type' => 'news', 'product_id' => 1, 'score_price' => 50, 'num' => 1],
                ['type' => 'video', 'product_id' => 5, 'score_price' => 4000, 'num' => 1],
            ];
            type:类型
                (news:资讯,
                video:视频)、
            product_id:对应的id，视频打赏及直播打赏传对应视频或直播的id、
                score_price:积分、
                num:数量
score_num(必) 使用积分数
mobile 购买人手机号
realname 购买人真实姓名
zone_id 购买人所在地区
address 购买人地址

返回数据(例):

{
    "message": "支付成功",
    "status": true,
    "forwardUrl": ""
}
