参数 值
uid(必) 当前登录用户uid
amount(必) 订单总价(元)
pay_way(必) ali或weixin
score_num(必) 使用积分数
items(必) 
            购买项，一个二维数组
            [
            ['type' => 'vip', 'product_id' => 1, 'price' => 500, 'num' => 1],
            ['type' => 'video_reward', 'product_id' => 2, 'price' => 1000, 'num' => 1],
            ['type' => 'live_reward', 'product_id' => 3, 'price' => 2000, 'num' => 1],
            ['type' => 'video', 'product_id' => 4, 'price' => 3000, 'num' => 1],
            ['type' => 'brand', 'product_id' => 5, 'price' => 4000, 'num' => 1],
            ['type' => 'brand_goods', 'product_id' => 5, 'price' => 4000, 'num' => 1],
            ];
            type:类型
                (vip:专版，video_reward:视频打赏,live_reward:直播打赏,video:视频,
                brand:品牌推介会下的品牌商品,brand_goods:品牌详情页的商品)、
            product_id:对应的id，视频打赏及直播打赏传对应视频或直播的id、
                price:价格,单位:元、
                num:数量
mobile 购买人手机号
realname 购买人真实姓名
zone_id 购买人所在地区
address 购买人地址
