品牌洽讯信息——tangjb

url链接：http://mt.wujie.com.cn/api/brand/consult

post

参数 值
size 返回条数，默认为0，为0则全部返回
id 默认为0，返回为一个月内的数据，如果大于0，则返回数据的id都大于此id

返回数据(例):

{ "message": [ { "uid": 1, "brand_id": 1, "id": 1, "created_at": "1478590236", "type": "quiz", "status": "pending", "nickname": "匿名用户12345", "brand_name": "测试", "created_at_format": "2016-11-08 15:30:36" }, { "uid": 1, "brand_id": 1, "id": 2, "created_at": "1478590236", "type": "intent_success", "status": "finish", "nickname": "匿名用户12345", "brand_name": "测试", "created_at_format": "2016-11-08 15:30:36" } ], "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
id 信息id
uid 用户id
brand_id 品牌id
type 类型，quiz:提问,intent:洽询,intent_success:洽询成功
nickname 用户昵称
brand_name 品牌名称
created_at_format 洽询时间
