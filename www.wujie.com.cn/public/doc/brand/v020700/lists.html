品牌列表——yaokai

url链接：http://mt.wujie.com.cn/api/brand/lists/_v020700

post

参数 值
uid 用户uid 未登陆传0
page 分页(默认1)
page_size 分页长度(默认8)
type guessYouLike:首页猜你喜欢 brandList:品牌列表(默认值) recommend:品牌推荐
zone_id 地区
investment_min 投资最小值
investment_max 投资最大值
categorys1_id 一级分类
categorys2_id 二级分类
orderby 排序 hot:人气最高 new:最新加入 investment_asc:金额从低到高 investment_desc:金额从高到低 joined:已参加过招商会
keywords 模糊搜索,品牌名称

参数说明（返回参数可补充）

参数 说明
(新)brand_summary 品牌列表的描述
id 品牌id
uid 品牌入驻用户
logo 品牌logo
name 品牌名称
investment_min 投资最小值
investment_max 投资最大值
keywords 关键字(array)
zone_name 所在地区名称
activity_id 关联活动id
category_name 一级分类
remark 标识语句
isRoadShow 是否参加过推介会，1:已参加,0:未参加
canJoin 是否可加盟，1:可加盟,0:不可加盟
