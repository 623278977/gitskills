获取商品信息——tangjb

url链接：http://mt.wujie.com.cn/api/brand/goods

post

参数 值
id(必) 商品id
type 默认live,live则和直播挂钩，brand则直接和品牌挂钩

返回数据(例):

{ "message": { "logo": "http://www.gpqqtx.com/uploads/allimg/151027/2234562339-5.jpg", "name": "品牌一号", "investment_min": "1.01", "investment_max": "2", "keywords": [ "凑足五个字" ], "zone_name": "杭州市", "category_name": "服装鞋包", "brand_id": 1, "id": 1, "price": 10, "num": 66, "code": "111-111", "status": "allow" }, "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
logo 品牌logo
detail 品牌图文详情
brand_summary 品牌描述
name 品牌名称
goods_title 商品标题
price 商品价格
code 商品编码
investment_min 品牌最小投资额，数字服务端已经格式化，单位:万元
investment_max 品牌最大投资额，数字服务端已经格式化，单位:万元
investment_arrange 品牌投资区间
zone_name 地区名称
category_name 类别名称
num 可购买数量
league 加盟政策
