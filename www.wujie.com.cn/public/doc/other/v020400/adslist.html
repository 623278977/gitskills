获得广告图——tangjb

url链接：http://mt.wujie.com.cn/api/ad/list/_v020400

post

参数 值
type 默认app_index_banner， app_index_banner:app首页的轮播, news:资讯头条页, find_brand:发现商机品牌页, index_reserved：首页预留,after_welcome:启动广告图,
page 分页,默认为1
page_size 规格,默认为5

返回数据(例):

{
  "message": [
    {
      "title": "启动页广告图",
      "type": "after_welcome",
      "link_url": "",
      "app_url": "",
      "stay_at": 0,
      "image": {
        "small": "http://www.wjsq3.com//attached/image/20161227/20161227171838_37015_480X800.jpg",
        "big": "http://www.wjsq3.com//attached/image/20161227/20161227171838_37015_720X1280.jpg"
      }
    }
  ],
  "status": true,
  "forwardUrl": ""
}

参数说明（返回参数可补充）

参数 值
image 图片地址,当类型为after_welcome时，返回一个数组,键名为 720X1280 或 480X800
link_url 跳转地址
title 标题
stay_at 停留时间,单位:秒
