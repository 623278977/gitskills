参数说明（返回参数可补充）

参数 说明
(新)score_price 积分形式价格
id 直播id
activity_id 相关联的活动id
live_url 直播频道号
subject 直播标题
description 描述
need_ticket 是否需要门票，0不需要，1需要
preview_time 预告时间，已经格式化(如：09/30 19:56)
begin_time 开始时间，时间戳
begin_time_format 开始时间，已经格式化(如：09/30 19:56)
end_time 结束时间，已经格式化(如：2017/09/30 19:56)
duration_hour 直播时长（小时）
list_img 列表页图片地址(string)
view 浏览量
status 是否已删除，0，未删除，1，已删除
chatroom_id 聊天室id
vip_id 所属专版id
subscribe 是否已订阅，1代表已订阅，0代表未订阅
is_being 是否正在直播，1代表正直播，0代表预告，-1代表已结束
share_score 分享送积分数
keywords 关键字
brand_name 品牌名称
min_price 票价
video_count 相关视频数量
length 长度
part_description 部分描述内容
