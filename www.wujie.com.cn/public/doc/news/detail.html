资讯详情——tangjb

url链接：http://mt.wujie.com.cn/api/news/detail

post

参数 值
id(必) 资讯id

返回数据(例):

{ "message": { "id": 1, "created_at": "1478586078", "relation_id": 1, "type": "brand", "detail": "1", "banner": "http://www.wjsq3.com/./1", "title": "1", "created_at_format": "2016-11-08 14:35", "brand": { "name": "测试", "category_name": "第一个分类" } }, "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
id 新闻id
created_at 发布时间时间戳
created_at_format 发布时间，已格式化
banner 新闻banner
title 新闻标题
author 新闻作者
detail 新闻详情
type 新闻分类，brand:品牌,none:无
share_mark 分享标识码
code 用于分享的唯一码
brand.name 品牌名称， 只有当type为brand时才返回
brand.category_name 品牌分类名称， 只有当type为brand时才返回
