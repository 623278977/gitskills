参数说明（返回参数可补充）

参数 说明
id 新闻id
created_at 发布时间时间戳
created_at_format 发布时间，已格式化
banner 新闻banner
title 新闻标题
author 新闻作者
detail 新闻详情
share_reward_unit 分享所得奖励的单位,score:积分，currency:无界币,none:无
share_reward_num 分享所得奖励的单位的个数
share_mark 分享标识
is_zan 是否当前用户点赞, 0 或 1
type 新闻分类，brand:品牌,none:无
当type=brand时:
增加 brand={json}
并且请求参数guess=1时
增加 guess_brands=[{json},{json}]
id 品牌ID
name 品牌名称
category_name 品牌分类名称
investment_min 品牌最小投资额度
investment_max 品牌最大投资额度
logo 品牌logo
created 品牌发布时间
keywords 品牌关键字数组
issuer 品牌发布人
details 品牌详情内容，已经去掉标签，纯文本内容
