资讯列表(商圈头条)——tangjb

url链接：http://mt.wujie.com.cn/api/news/list

post

参数 值
page 分页(默认1)
page_size 分页长度(默认15)

返回数据(例):

{ "message": [ { "id": 1, "is_hot": 0, "is_recommend": 0, "relation_id": 1, "type": "brand", "detail": "1", "logo": "http://www.wjsq3.com/./1", "title": "1", "brand": { "name": "测试", "category_name": "第一个分类" } }, { "id": 2, "is_hot": 0, "is_recommend": 0, "relation_id": 1, "type": "brand", "detail": "2", "logo": "http://www.wjsq3.com/./2", "title": "2", "brand": { "name": "测试", "category_name": "第一个分类" } }, { "id": 3, "is_hot": 0, "is_recommend": 0, "relation_id": 0, "type": "none", "detail": "3", "logo": "http://www.wjsq3.com/./3", "title": "3" } ], "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
id 新闻id
is_hot 是否热门，1:热门，0:非热门
is_recommend 是否推荐，1:推荐，0:非推荐
logo 新闻logo
title 新闻标题
detail 新闻详情
type 新闻分类，brand:品牌,none:无
brand.name 品牌名称， 只有当type为brand时才返回
brand.category_name 品牌分类名称， 只有当type为brand时才返回
