专版视频列表——tangjb

url链接：http://mt.wujie.com.cn/api/vip/videos

post

参数 值
vip_id(必) 专版id
category 类别，默认为空
sort 排序，不传则综合排序
filter 筛选，(maker:本地商圈、recommend:小编推荐、hot:热门视频)
uid 当参数filter为maker时，该参数必传

返回数据(例):

{ "message": [ { "id": 200036, "subject": "subject29", "image": null, "type": "金融产品", "is_recommend": 0, "description": "", "view": 0, "favor_count": 0 }, { "id": 200035, "subject": "3333", "image": null, "type": "金融产品", "is_recommend": 0, "description": "", "view": 0, "favor_count": 0 } ], "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
id 视频id(string)
subject 视频标题(string)
image 视频图片地址(string)
type 视频类别(string)
is_recommend 是否推荐1，代表推荐，0代表未推荐(string)
description 视频描述
view 观看数
favor_count 收藏数
