专版购买记录列表——tangjb

url链接：http://mt.wujie.com.cn/api/vip/record

post

参数 值
uid(必) 当前登录用户uid
vip_id 若是传递了此参数，则返回用户在该专版下的购买记录

返回数据(例):

{ "message": [ { "name": "测试数据", "id": 1, "start_time": "2016/05/05", "end_time": "2016/05/05", "package_name": "6个月专版会员套餐", "is_expire": 1 }, { "name": "测试数据2", "id": 2, "start_time": "2016/05/05", "end_time": "2016/05/05", "package_name": "1个月专版会员套餐", "is_expire": 1 } ], "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
name 专版名称(string)
id 专版id(string)
start_time 生效时间(string)
end_time 过期时间(string)
package_name 套餐名称(string)
is_expire 是否已经过期， 1：已过期，0：未过期
is_using 是否正在使用中，1：正在使用，0：不是正在使用
