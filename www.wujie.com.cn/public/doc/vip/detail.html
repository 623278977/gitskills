专版详情——tangjb

url链接：http://mt.wujie.com.cn/api/vip/detail

post

参数 值
vip_id 专版id
uid 登录用uid,未登录传0
attach 传1将会返回该专版下的活动数量，视频数量，直播数量，默认0
agreement 传1将会返回该专版下的会员权益说明，默认0
package 传1将会返回该专版下的套餐信息说明，默认0

返回数据(例):

"message": { "name": "测试数据", "detail": "good", "poster": "attached/image/20160809/20160809174129_73341.jpg", "agreement": "很不错的套餐", "subtitle": "", "activity_count": 2, "video_count": 3, "live_count": 2, "package": [ { "is_recommend": 1, "period": "6个月", "name": "asdf", "price": "500.00", "status": 1 }, { "is_recommend": 1, "period": "6个月", "name": "hahah", "price": "500.00", "status": 0 }, { "is_recommend": 1, "period": "6个月", "name": "563", "price": "500.00", "status": 0 }, { "is_recommend": 1, "period": "3个月", "name": "好套餐", "price": "1000.00", "status": 0 } ] }, "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
id 专版id(string)
name 专版名(string)
poster 海报图片地址(string)
detail 专版详情(string)
groupid 环信群聊id(string)
subtitle 副标题(string)
status 该专版状态，1：可用，0：不可用
activity_count 该专版下的活动数目，当参数attach为1时才返回
video_count 该专版下的视频数目，当参数attach为1时才返回
live_count 该专版下的直播数目，当参数attach为1时才返回
agreement 该专版下的会员权益，当参数agreement为1时才返回
is_valid 当前登录用户是否为该专版的有效会员，当参数agreement为1时才返回
package.is_recommend 该套餐是否推荐，当参数package为1时才返回
package.period 该套餐的周期如3个月、1年，当参数package为1时才返回
package.name 该套餐的名称，当参数package为1时才返回
package.price 该套餐的价格，当参数package为1时才返回
package.status 该套餐的状态，为1则表示可用，为0则表示不可用，当参数package为1时才返回
package.id 该套餐的id
