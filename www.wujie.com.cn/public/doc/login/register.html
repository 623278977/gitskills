用户注册——tangem

url链接：http://mt.wujie.com.cn/api/login/register

post

参数 值
username(必) 手机号
password(必) 密码
nickname(必) 昵称
is_outh(必) 是否是第三方注册 默认0（不是） 如果是 传1
zone_id 地区id
industry_id 关注行业id 数组
outh_id 第三方登陆的access_token 如果是第三方注册必填
type 第三方登陆必填 qq传qq 微信传wx 微博传wb
code 分享唯一标识
is_invited 是不是通过邀请注册默认0不是，1是
captcha_id 若是邀请注册则，验证码id必传
captcha_value 若是邀请注册则，验证码值比传
reg_method 注册方式 pass:常规注册 sms:短信注册(老用户直接登录返回用户信息,新用户进行后续注册步骤) setinfo:完善资料
