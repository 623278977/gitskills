邀请注册——tangem

url链接：http://mt.wujie.com.cn/api/login/inviteregister

post

参数 值
username(必) 手机号
password(必) 密码
nickname(必) 昵称
code(必) 分享唯一标识
captcha_id(必) 若是邀请注册则，验证码id必传
captcha_value(必) 若是邀请注册则，验证码值比传
nation_code 默认中国:86, 美国:1

返回数据(例):

{ "message": { "score": 100, "nickname": "nickname" }, "status": true, "forwardUrl": "" }

参数 说明

nickname 用户名
uid 用户uid
