活动邀请——tangem

url链接：http://mt.wujie.com.cn/api/activity/invite

post

参数 值
uid(必) 用户uid
activity_id(必) 活动id(string)
maker_id(必) ovoid(string)

返回数据(例):

{ "message": { "vip_id": 1, "need_invite_num": 5, "has_invited_num": 0, "tickets": { "subject": "2", "is_receivable": 1, "type": "现场票", "begin_time": "2016-12-31 00:00" } }, "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
vip_id 专版ID
need_invite_num 需邀请人数
has_invited_num 已经邀请人数
tickets.subject 活动标题
tickets.is_receivable 是否可领取（0不可 1可以领取 2已经领取）
tickets.type 门票类型
tickets.end_time 活动结束时间
tickets.begin_time 活动开始时间
tickets.remark 备注
