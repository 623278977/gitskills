使用积分购买活动或者直播——tangjb

url链接：http://mt.wujie.com.cn/api/activity/apply-and-pay/_v020700

post

参数 值
uid(必) 用户uid
activity_id(必) 活动id(string)
ticket_id(必) 门票id
score_num(必) 使用积分数
maker_id 空间id(现场票必填)
company 公司
job 职务
product 活动(直播)标题
body 活动(直播)描述,可以传和活动标题一样的内容
name 参会人姓名
tel 参会人手机号
path 报名通道,app:手机app，html5:分享出去的页面,默认app
code 手机验证码，只有当path参数为html5时，才需要传。

返回数据(例):

{ "message": "jp1498015863no", "status": true, "forwardUrl": "" }

参数说明（返回参数可补充）

参数 说明
order_no 返回订单号，供签名
is_register 是否已经注册，该参数只有当报名的通道为html5时才返回，1代表已经注册，0代表未注册
