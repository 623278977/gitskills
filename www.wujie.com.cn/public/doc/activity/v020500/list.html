活动列表——pengwt

url链接：http://mt.wujie.com.cn/api/activity/list/_v020500

post

参数 值
page 分页(默认1)
pageSize 分页长度(默认15)
maker_id ovo中心id(不传取所有活动)
type 1将来活动(默认)2过去活动 0所有活动
vip_id 专版id(非必传)
keywords 关键词
hotwords 热门关键字

返回数据(例):

参数说明（返回参数可补充）

参数 说明
id 活动id(string)
subject 活动标题(string)
view 浏览量(string)
list_img 列表页图片地址(string)
detail_img 一张详情图片地址(string)，isOver==-1时才存在该值
is_recommend 是否推荐0否 1是
begin_time 开始时间
price 门票价格
isOver 活动是否已经结束 1:已经结束 0:未开始 -1:正在进行
city 举行该场活动的城市
type 举行该场活动的城市的类型（主办方，协办方），当有多个时，以@隔开,与上面的city字段对应(string或array)
url 当前活动的链接
is_vip 是否为专版活动(1是，0否)
is_hot 是否为热门活动(1是，0否)
vip_id 专版ID
ticket_num 门票数量
is_shareable 是否可分享(1是，0否)
share_score 分享送积分数
keywords 关键字
brand_name 品牌名称
begin_time_format 开始时间,对应020500版本
live_support 直播支持 1 是 0 否
sign 正在报名中 1 是 0 否
dataCount 数据总数
