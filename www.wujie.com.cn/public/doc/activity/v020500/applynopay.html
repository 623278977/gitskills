活动报名不支付——tangjb

url链接：http://mt.wujie.com.cn/api/activity/apply-no-pay/_v020500

post

参数 值
uid(必) 用户uid
activity_id(必) 活动id(string)
company 公司
job 职务
ticket_id(必) 门票id
maker_id(必) 空间id
cost(必) 价格,免费票传0
product（必） 活动标题
body（必） 活动描述,可以传和活动标题一样的内容
pay_way（必） ali或weixin或none， 免费传none
name（必） 参会人姓名
tel（必） 参会人手机号
score_num 使用积分数
path 报名通道,app:手机app，html5:分享出去的页面,默认app
code 手机验证码，只有当path参数为html5时，才需要传。
share_mark 分享出去的报名需要带上这个参数

返回数据(例):

{"message":"jv1465203198qd","status":true,"forwardUrl":""}

参数说明（返回参数可补充）

参数 说明
order_no 返回订单号，供签名
is_register 是否已经注册，该参数只有当报名的通道为html5时才返回，1代表已经注册，0代表未注册
activity_sign_id 活动报名表id
