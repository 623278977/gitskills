参数说明（返回参数可补充）

参数 说明
id 活动id
subject 活动名称
is_vip 是否专版活动 1 是 0 否
vip_name 专版名称
vip_id vip的id
is_shareable 是否可以分享：1可以， 0不可以
is_collect 是否已经收藏活动：1已经收藏， 0未收藏
share_image 分享图标
can_buy 是否可以购买专版 1 可以 0 否
detail_img 活动详情图,返回数组
begin_time 活动开始时间
end_time 活动结束时间
begin_time_content.begin_time 活动开始时间
begin_time_content.time_explain 活动时间说明文字
begin_time_content.period 活动时长
activity_location 活动地点
activity_location_arr.id ovo的id
activity_location_arr.subject ovo名称
activity_location_arr.address ovo地址
activity_location_arr.tel ovo电话
activity_location_arr.zone_id ovo的zone_id
activity_location_arr.zone ovo的地区名
activity_location_arr.description ovo的描述
min_ticket_price 最便宜的票价格
min_ticket_price_type 最便宜的票类型
ticket_id 免费票 ticket_id ; 其他票 0
sign_count 活动数
view_count 浏览数
zan_count 赞数
comment_count 评论数
share_count 分享次数
hot_count 热度
site_need_pay 是否具有需要付费的现场票，1有，0没有
description 活动详情
brand.summary 品牌概述
brand.id 品牌id
brand.uid 品牌入驻用户
brand.logo 品牌logo
brand.name 品牌名称
brand.investment_min 投资最小值
brand.investment_max 投资最大值
brand.keywords 关键字(array)
brand.zone_name 所在地区名称
brand.activity_id 关联活动id
brand.category_name 一级分类
brand.remark 标识语句
zans.avatar 点赞用户头像
share_reward_unit 活动分享所得奖励的单位,score:积分,currency:无界币，none:什么都不奖励
share_reward_num 活动分享所得奖励的个数
banners 活动横幅(数组)
is_praise 是否点赞，1已点赞，0未点赞
share_mark 分享识别码
code 用于分享的唯一码
