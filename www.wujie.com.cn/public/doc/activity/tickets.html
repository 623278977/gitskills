活动门票——tangjb

url链接：http://mt.wujie.com.cn/api/activity/tickets

post

参数 值
id 活动id(必)
uid 登录用户uid，未登录传0(必)

参数说明（返回参数可补充）

参数 说明
id 门票id(string)
activity_id 活动id(string)
type 门票类型 1现场票 2直播票
price 价格(string)
intro 门票介绍(string)
num 门票总数量(string)
remark 备注(string)
left 该类型门票还剩下多少(num)
is_over 该场活动是否已经结束，1：已结束，0：未结束(num)
is_share 该门票是否允许通过分享活动 1:可分享 0：不可分享
vip_id 该门票所属于的活动是否为专版活动， 如果是则返回该专版的id， 如果不是则返回0
is_vip 当前登录用户是否是该活动的有效会员 1:代表是，0:代表不是
subject 活动名称
begin_time 活动开始时间
end_time 活动结束时间
maker_id 活动主办场地空间id
address 活动主办场地空间地址
maker_subject 活动主办场地空间名称
